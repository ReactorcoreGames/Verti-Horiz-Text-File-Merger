Vertical Stacking Help

×

What it does:

The Vertical Merger combines multiple text files into one, joining them one after another in alphabetical filename order.

How to use:

  1. Upload multiple text files by dragging them onto the upload area or clicking "Select Files"
  2. Files will be automatically sorted alphabetically by filename
  3. Each file's content will start on a new line
  4. Preview the result in the text area
  5. Copy to clipboard or download the merged file

Horizontal Append Help

×

What it does:

The Horizontal Appender combines two text files line by line, merging corresponding lines from each file.

How to use:

  1. Upload a base file (left slot) and an addon file (right slot)
  2. Choose whether addon lines come before or after base lines
  3. Enable "Repeat addon lines" to loop addon content if it's shorter
  4. Enable "Remove extra spaces" to clean up formatting
  5. Preview the result and copy or download

Options:

  • Addon after base: Places addon text after base text on each line
  • Repeat addon: If addon has fewer lines, it loops from the beginning
  • Remove extra spaces: Converts multiple spaces to single spaces
